1. To get pixel library in the terminal type 
	go get github.com/faiface/pixel/.. 

2. To get the beep library, in the terminal type:
	go get github.com/faiface/beep/

3. To run the program in the terminal type
	go run berzerk.go

4. Game Instructions:
	Left arrow => Move left
	Right arrow=> Move Right
	Down arrow => Move Down
	Up arrow => Move Up
	Spacebar => Shoot

5. To view go doc in the Terminal Type:
	godoc berzerkGame/gameDoc
